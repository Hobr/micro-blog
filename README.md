# Hobr's Blog

## 命令

```bash
# 安装依赖
pnpm install

#  启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 预览构建结果
pnpm preview

# 使用 Astro CLI
pnpm astro -- --help

# 检查代码
pnpm lint

# 格式化代码
pnpm format
```
